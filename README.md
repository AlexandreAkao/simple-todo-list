## How To Use

Install with npm or yarn

```bash
# Clone this repository
$ git clone https://github.com/AlexandreAkao/simple-todo-list.git

# Go into the repository
$ cd simple-todo-list

# Install dependencies
$ npm install

$ npm run start
```
